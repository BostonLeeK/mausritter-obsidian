# Mausritter

Настільна рольова гра про мишей-шукачів пригод у мініатюрному фантастичному світі. Візьміть торчинку, меч і рюкзак — і вирушайте досліджувати темні ліси, печери та небезпечні краї.

Цей Obsidian-проєкт призначений для данжен-майстра: він дозволяє швидко ознайомитися зі сценаріями, локаціями, квестами та ворогами та зручно проводити гру.
